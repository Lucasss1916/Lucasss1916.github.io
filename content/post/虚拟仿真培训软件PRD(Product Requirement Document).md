# 1 项目背景
目前电网针对多种不同型号开关柜以及不同的电子屏存在培训成本过高、无法统一标准以及无法量化培训成果的问题。
# 2 产品目标 
1. 提升培训效率
2. 统一培训标准
3. 降低培训成本
4. 降低操作错误率
# 3 用户角色
1. 学员
2. 管理员
# 4 核心训练场景
进入场景 - 场景训练 - 操作反馈 - 评分

## 5.2 评分系统
### 5.2.1 评分规则
1. 单人考核：每个开关柜都有一百分为满分，考核项目的数量也有差别，分数的分配提前在系统中配置好。
2. 多人考核：单个柜子的满分为每人一百分，一共有三名用户进行操作，玩家一为检查操作，找出相应的问题，100分平均分配给每一个问题，全找出得到满分;玩家2,根据玩家一找出的问题进行修改，100分平均分配给每一项;玩家三为检查工作，得分为玩家一玩家二的总分之和除以2。
