# 1. 项目背景
目前电网针对多种不同型号开关柜以及不同的电子屏存在培训成本过高、无法统一标准以及无法量化培训成果的问题。
# 2. 产品目标
1. 提升培训效率
2. 统一培训标准
3. 降低培训成本
4. 降低操作错误率
# 3. 用户角色
1. 学员
2. 管理员
# 4. 核心训练场景
进入场景 - 场景训练 - 操作反馈 - 评分 
# 5. 产品功能模块拆解
1. 仿真训练端
2. 评分系统
3. 管理后台
4. VR体验端
